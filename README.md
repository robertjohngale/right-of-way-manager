# Right of Way Manager Widget

ArcGIS Experience Builder widget for managing centerlines and right-of-way polygons.

## Features

- Draw or select multiple polyline centerlines
- Configure left/right widths for ROW buffers
- Generate ROW polygons with accurate geometry
- Vertex analytics with bend angles and bearings
- Export lines and polygons to GeoJSON
- Export vertex analytics to CSV
- Manage multiple projects with metadata tables

## Installation

Copy this widget folder to:
`client/your-extensions/widgets/right_of_way_manager`

Then restart Experience Builder Developer Edition.
